# 📂 Automated File Management System

## 🚀 Project Overview
This project automates the workflow of document archiving by creating a seamless pipeline between *Gmail* and *Google Drive*. It eliminates manual downloads and ensures all business attachments are organized in cloud storage in real-time.

## 🛠️ Tools Used
* *Zapier:* Automation Logic
* *Gmail:* Trigger Source
* *Google Drive:* Destination Cloud Storage

## ⚙️ How it Works
1. *Trigger:* Monitors a specific Gmail account for new attachments.
2. *Action:* Zapier retrieves the file and metadata.
3. *Outcome:* Automatically uploads the file to a designated folder in Google Drive.

## ✅ Key Benefits
* *Efficiency:* 100% reduction in manual file sorting.
* *Accuracy:* No documents are missed or lost in transit.
